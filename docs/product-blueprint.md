# Product Blueprint (v1)

## Primary User Outcome
User gets clear, non-judgmental suggestions on how to save more using existing
cards, offers, and spending habits.

## End-to-End User Flow
1. Install app.
2. Upload 3, 6, or 12 months of statements (mobile OCR supported).
3. Confirm detected cards and add missing/inactive cards.
4. Enter salary and recurring fixed commitments.
5. Receive immediate dashboard:
   - cashback earned,
   - missed opportunity estimate,
   - top 3 actions.
6. Ongoing:
   - push notifications/quick add where available,
   - monthly statement upload reminder,
   - dedupe + reconciliation,
   - refreshed recommendations.

## Core Home Widgets
- Cashback earned this month.
- Missed rewards estimate.
- Best card now (category-aware).
- Expiring offer alerts.
- Search bar: "What are you buying?"

## AI Assistant Rules
- Always provide options, not commands.
- Show tradeoffs (higher rewards vs lower utilization vs lower interest risk).
- Add assumptions and confidence level for each option.
- Never present output as licensed advice.

## Travel Query Flow
Input: user asks for a trip (for example: Miami).
Output:
- best card mix for flight and hotel,
- likely cashback/points stack,
- potential portal add-on opportunities,
- estimated net benefit range.
